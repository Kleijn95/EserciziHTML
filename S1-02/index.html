Come creare un template per i post

di Mario Rossi - pubblicato il 25-10-2020

La prima cosa da fare é suddividire in blocchi logici la pagina, identificando le parti comuni da applicare a tutte le pagine
    Le varie aree saranno quindi suddivisi in:
  • Contenuti di instenziona del tag <header>
  • Contenuti centrali variabili del tag <main>
  • Contenuti di fine pagine comuni, da inserire nel tag <footer>

Formattazione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con i CSS. Perciò i passi saranno:
  1. Progettare i markup assegnando ad ogni specifico testo il tag appropriato
  2. Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  3. Appliciare gli stili CSS
    4. L'applicazione degli stili CSS é cresciuta negli anni fino a raggiungere la diffusione pressoché totale.
